# 开源趋势与影响力分析平台 - 技术文档

## 文档信息

- **项目名称**：开源趋势与影响力分析平台
- **项目描述**：一站式开源项目数据分析与智能决策平台



## 摘要

开源趋势与影响力分析平台是一个基于 Web 的开源项目数据分析平台，旨在为用户提供全面的开源项目影响力分析、技术趋势预测和智能数据查询功能。平台整合了 GitHub 数据源，通过数据可视化和分析算法，帮助用户做出数据驱动的决策。本文档详细描述了项目结构、系统的架构设计、技术实现、功能模块和项目运行方式。



## 系统概述

### 项目背景

随着开源生态的快速发展，开发者需要高效工具来评估项目质量、跟踪技术趋势和发现潜在机会。因此，我们开发了这个开源趋势与影响力分析平台，旨在提供开源形式分析、助力用户构建数据驱动的开源相关决策。

### 主要功能

- **项目影响力分析**：基于多维度指标评估项目影响力
- **技术趋势预测**：分析技术栈演变和未来发展趋势
- **智能问答系统**：自然语言查询数据能力
- **数据可视化**：丰富的图表展示和交互界面



## 项目结构

project项目文件夹结构如下：

* raw_data：待处理的原始数据文件夹
* data：处理后的数据csv文件夹
	* projects_complete.csv：300个开源项目的完整数据，包含29个数据字段
	* influence_ranking.csv：项目影响力排名数据
	* language_trends_detailed.csv ：编程语言时序数据
	* keyword_trends.csv：技术关键词时序数据
	* technology_maturity.csv：技术成熟度评估数据
* data_processing：数据预处理相关python文件，raw_data->data
* index.html：网站首页
* style.css：网站通用样式
* influence-analysis：项目影响力分析页面
* trend-analysis：技术趋势分析与预测页面
* projects：项目选择页面与项目主页
* languages：编程语言选择界面与主页
* technology：技术关键词选择界面与主页

## 功能模块

### 1. 项目影响力分析模块

**功能描述**：评估开源项目的综合影响力

**核心指标**：

- OpenRank 分数
- 星标数 (Stars)
- Fork 数
- 贡献者数量
- 活跃度评分

**实现方式**：

- 计算方法：加权算法
- 可视化：排名列表、雷达图

### 2. 技术趋势分析模块

**功能描述**：分析技术栈发展趋势

**核心功能**：

- 技术热度跟踪
- 趋势预测模型
- 关键词分析
- 时间序列分析

### 3. 智能问答模块 (SQLBot)

**功能描述**：自然语言数据查询

**技术实现**：

- 预定义查询模板
- 关键词匹配
- SQL生成
- 结果格式化

### 4. 数据检索模块

**功能描述**：分类浏览数据

**子模块**：

- 编程语言分析
- 技术关键词分析
- 项目详情查看



## 数据处理流程

### 数据采集流程

- 数据来源于OpenRank竞赛参考资料，为GitHub开源项目数据

### 数据处理流程

1. **数据清洗**：
	- 移除重复记录
	- 处理缺失值
	- 数据类型转换

2. **特征工程**：
	- 计算派生指标
	- 标准化处理
	- 分类编码

3. **数据聚合**：
	- 生成完整的projects.csv文件，包含所有29个维度的原始数据指标
	- 为各功能模块生成支撑CSV文件

### 数据可视化流程

1. **前端数据加载**：异步加载 CSV/JSON 数据
2. **图表渲染**：ECharts 配置和渲染
3. **交互处理**：用户操作响应和数据更新



## 安装与部署

### 环境要求

- **操作系统**：Windows 10+ / macOS 10.15+ / Linux Ubuntu 18.04+
- **Python**：3.8 或更高版本
- **浏览器**：Chrome 90+ / Firefox 88+ / Safari 14+
- **内存**：至少 4GB RAM
- **存储**：至少 1GB 可用空间

### 安装步骤

1. **克隆项目**：

	```bash
	git clone <repository-url>
	cd OpenRank/Program/Project
	```

2. **安装依赖**：

	```bash
	pip install -r requirements.txt
	```

3. **启动前端服务**：

	- 使用 VS Code Live Server 插件 
	- python -m http.server 8000
	- 在浏览器中打开 http://localhost:8000
