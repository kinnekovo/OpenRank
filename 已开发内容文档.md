# OpenInsight项目已开发内容文档

## 1. 项目概述
OpenInsight是一个一站式项目数据分析与智能决策平台，整合项目影响力分析、技术趋势预测和智能问答能力，为项目评估与决策提供全方位数据支撑。

## 2. 项目结构

```
├── index.html                   # 首页
├── influence-analysis/          # 影响力分析模块
│   ├── index.html              # 影响力分析主页面
│   └── main.js                 # 影响力分析功能实现
├── trend-analysis/              # 技术趋势分析模块（待完善）
├── maxkb-qa/                   # MaxKB智能问答模块（待完善）
├── data/                       # 数据文件目录
│   ├── dashboard_summary.csv      # 平台概览数据
│   ├── influence_ranking.csv      # 项目影响力排名
│   ├── keyword_trends.csv         # 关键词趋势数据
│   ├── language_trends_detailed.csv # 语言趋势详细数据
│   ├── projects_complete.csv      # 完整项目数据
│   └── technology_maturity.csv    # 技术成熟度数据
├── data_processing/            # 数据处理模块
│   ├── github_data_loader.py      # GitHub数据加载器
│   ├── generate_complete_projects.py # 生成完整项目数据
│   ├── config.py                  # 配置文件
│   └── ...                       # 其他数据处理脚本
├── raw_data/                   # 原始数据目录
│   └── top_300_metrics/          # GitHub Top300项目指标
└── common/                     # 通用资源
    ├── bootstrap-icons.min.css   # Bootstrap图标
    ├── echarts.min.js            # ECharts可视化库
    └── papaparse.min.js          # CSV解析库
```

## 3. 核心功能模块

### 3.1 首页 (index.html)
- **导航栏**：固定顶部，包含首页、影响力分析、技术趋势分析、MaxKB智能问答四个主要入口
- **全屏Banner**：展示平台核心价值主张
- **功能卡片**：快速访问各个核心模块

### 3.2 影响力分析模块 (influence-analysis/)

#### 3.2.1 平台概览
- 总项目数统计
- 总Star数统计
- 平均影响力评分
- 支持的编程语言数量

#### 3.2.2 项目影响力排名
- **可视化图表**：使用ECharts展示Top30项目的影响力得分趋势
- **排名表格**：展示详细项目排名信息，包含项目名称、影响力得分、语言等
- **展开/收起功能**：可切换显示前15条或全部项目数据
- **语言筛选**：支持按编程语言筛选项目

#### 3.2.3 语言分布分析
- 展示不同编程语言的项目数量分布
- 支持交互式图表展示

### 3.3 数据处理模块 (data_processing/)

#### 3.3.1 GitHub数据加载器
- 加载GitHub Top300项目的原始数据
- 支持多种指标数据：元数据、Star数、活跃度、OpenRank等
- 提供统一的数据访问接口

#### 3.3.2 数据生成与处理
- 生成完整项目数据CSV
- 计算影响力得分
- 处理语言识别与映射
- 生成各种分析报告
